Detailed Description

Header file defining types and constants for Modbus protocol implementation.

This header file defines various types, constants, enums, and structures used in the implementation of the Modbus protocol. It includes definitions for Modbus RTU buffer sizes, maximum data quantities, offsets for Modbus frame parameters, error codes, communication states, flags, and data types like Modbus messages and function handlers.

The defined constants ensure compatibility with Modbus standard specifications, facilitating the development of Modbus-compliant applications. The file also includes enums for handling various return codes and states, and data structures for message handling.
