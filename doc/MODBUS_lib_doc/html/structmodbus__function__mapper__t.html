Detailed Description

Structure representing a mapping between Modbus function codes and their handlers.

This structure is used to map Modbus function codes to their corresponding handler functions. Each function code is associated with a function pointer that defines the action to be taken when a message with that function code is received.

The fun_code member holds the Modbus function code, and the fun_code_action member holds the function pointer to the handler that processes the corresponding Modbus message.

Example usage:

modbus_function_mapper_t mapper = {
.fun_code = READ_HOLDING_REGISTERS,
.fun_code_action = handle_read_holding_registers
};
