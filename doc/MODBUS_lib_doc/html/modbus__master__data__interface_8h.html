Defines functions for Modbus master data operations and error handling.

This header file declares functions for Modbus master data operations and error handling. It includes functions for reading and writing coils (discrete outputs), reading and writing holding registers (Holding Registers), reading discrete inputs, and handling exceptions for coil and register reads. Additionally, it provides functions to handle timeouts and communication errors specific to Modbus master operations.

The functions are designed to interact with a Modbus slave device identified by its address (slave_adr) and specific data addresses (data_adr). They utilize types defined in modbus_public_type.h for consistency and clarity in data handling.

Note
Implement these functions according to the Modbus protocol specifications and the requirements of your Modbus master application. Handle errors and exceptions appropriately to ensure reliable communication with Modbus slave devices.
