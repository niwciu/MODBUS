Detailed Description

Defines public types and enums used in Modbus communication.

Author
niwciu (niwci.nosp@m.u@gm.nosp@m.ail.c.nosp@m.om)
Date
2024-05-30

This header file defines various enums and typedefs that are essential for Modbus communication and error handling. It includes enums for Modbus modes (RTU and unknown), parity settings, Modbus exception codes, Modbus function codes, and other types used in Modbus data communication.

This file serves as a public interface to define Modbus-specific types, making the codebase more readable and maintainable by providing clear abstractions for Modbus-related concepts.
